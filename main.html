Úvod do KBB & Malware

Typy hackerů

White Hat

Etičtí hackeři. Identifikují a opravují zranitelnosti se svolením organizace.

Gray Hat

Bez kriminálních úmyslů, ale nabourávají se bez souhlasu — aby upozornili na chybu.

Black Hat

Kyberzločinec. Jedná nelegálně pro vlastní prospěch.

Týmy etických hackerů

Red Team

Ofenzivní tým. Hledá zranitelnosti, provádí penetrační testování.

Blue Team

Defenzivní tým. Ochraňuje sítě, kontroluje logy, tech support.

Purple Team

Zajišťuje komunikaci a spolupráci mezi Red a Blue týmem.

Druhy malwaru

Virus

Šíří se vkládáním do jiných spustitelných souborů či dokumentů.

Červ (Worm)

Šíří se přes sítě využitím zranitelností OS. Nepotřebuje hostitele.

Trojský kůň

Tváří se jako užitečný software, ale skrytě škodí.

Adware / Spyware

Zahlcuje reklamou / špehovává. Keylogger (klávesy), Screenlogger (obrazovka).

Ransomware

Zašifruje disk a požaduje výkupné. Šifrovací nebo zamykací varianta.

Speciální typy & analýza

Polymorfický

Mění vlastní kód při každém běhu — obtížná detekce antivirem.

File-less

Existuje pouze v RAM. Nepíše soubory na disk.

Statická analýza

Bez spuštění kódu. Nástroj: virustotal.com

Dynamická analýza

Chování při spuštění v izolaci. Nástroj: any.run

Ransomware skupiny

REvil

Zaniklá skupina — poskytovala RaaS (Ransomware as a Service).

Lazarus

Severokorejská státem sponzorovaná skupina.

Cyber Partisans

Běloruská hacktivistická skupina. Požadovala propuštění 50 politických vězňů.

Sítě & Síťová bezpečnost

Základní pojmy

LAN / WAN / Internet

LAN = lokální síť. WAN = propojení více sítí. Internet = všechny sítě světa.

Switch vs Router

Switch propojuje v LAN. Router směruje provoz mimo LAN do WAN.

Server – Client – P2P

Server poskytuje služby. P2P = bez centrálního serveru.

Fyzická média

Měděný kabel (UTP/STP)

Elektrické signály, koncovka RJ-45. UTP náchylnější na rušení, STP má stínění.

Optický kabel

Přenos světla — tenká křehká vlákna, vyšší rychlosti a delší vzdálenosti.

Koaxiální

Dnes zastaralý.

Adresace

IPv4 (32-bit)

4 oktety. ~4.3 mld adres → dochází. Privátní: 10.x, 172.16–31.x, 192.168.x

IPv6 (128-bit)

340 undecillionů adres. Hex formát. Postupná náhrada IPv4.

Loopback

127.0.0.1 (localhost) — zařízení posílá data samo sobě.

MAC adresa

Fyzická adresa NIC. 48-bit hex. 1. část = výrobce, 2. část = karta.

Protokoly

DNS

Překlad domén na IP. Hierarchie: TLD → doména → subdoména.

DHCP

Dynamické přidělování IP adres. Obvykle na routeru.

TCP vs UDP

TCP = spolehlivý (kontrola ztrát). UDP = rychlý — streaming.

HTTP/S · SSH · Telnet

HTTP:80, HTTPS:443, SSH:22 (šifrované CLI), Telnet:23 — zastaralý.

Kryptografie

Šifrování

Symetrické

1 klíč pro šifrování i dešifrování. Rychlejší. AES, Serpent, Twofish.

Asymetrické

Veřejný klíč šifruje, soukromý dešifruje. RSA, CRYSTALS-Kyber (post-kvantové).

Šifrování disku

LUKS (Linux, open-source). VeraCrypt (univerzální, all-OS).

Hash

Co je hash?

"Otisk prstu" dat. Jednosměrná funkce — nelze dešifrovat zpět.

Příklady

MD5 (zastaralý), SHA-512, Blake2

Crackování

Wordlisty, Rainbow Tables. Nástroje: Hashcat, John the Ripper.

ukázka MD5
ahoj → 79c2b46ce2594ecbcb5b73e928345492
Kódování & Obfuskace

Kódování (Encoding)

Přepis do jiné formy. Není šifrování. Base64, Morse, Braille.

Obfuskace

Znesnadňuje čitelnost kódu pro člověka, bez vlivu na funkci.

Digitální podpis

Odesílatel podepisuje hash soukromým klíčem. Příjemce ověří veřejným.

Signal & Algoritmy

Double Ratchet

Rotace klíčů — každá zpráva má jiný klíč. Forward secrecy + post-compromise security.

Diffie-Hellman

Bezpečná výměna symetrických klíčů přes nezabezpečený kanál.

Post-kvantová krypto

Ochrana před kvantovými počítači. Ohrožena hlavně asymetrická krypto (RSA).

Kryptoměny & Blockchain

Blockchain

Decentralizovaná databáze transakcí. Veřejně dohledatelné (Bitcoin, Ethereum).

Monero

Jediná plně anonymní kryptoměna.

Využití v kriminalitě

Ransomware skupiny požadují krypto kvůli anonymitě.

Kybernetické útoky & OSINT

⚠ Obsah slouží pouze pro vzdělávací účely. Útoky na cizí systémy bez povolení jsou nelegální.
Zranitelnosti

Exploit

Program/skript využívající programátorskou chybu.

CVE

Identifikátor zranitelnosti. Př: log4shell = CVE-2021-44228

CWE

Kategorizace typů slabostí. CWE-79 = XSS.

CVSS

Skóre závažnosti zranitelnosti (verze 4.0).

Zero-day

Exploit pro neznámou chybu — žádný patch neexistuje.

MITRE ATT&CK

Framework pro analýzu útočných technik. → attack.mitre.org

Typy útoků

Phishing

Podvodné emaily imitující banku. Poznáme podle domény (@cs0b.cz). Nejčastější Initial Access (MITRE).

DoS / DDoS

Zahlcení serveru. DDoS využívá botnet (síť ovládaných PC) — těžko blokovat.

XSS

Cross-Site Scripting — vložení JS do stránky. Cíl: cookies, localStorage.

SQLi

SQL Injection — vložení kódu do dotazu. Cíl: data z databáze.

OSINT

OSINT Framework

Nástroje pro sběr dat z veřejných zdrojů. → osintframework.com

Shodan.io

Vyhledávač pro IoT zařízení, kamery, servery na internetu.
